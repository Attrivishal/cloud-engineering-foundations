# Elastic IP (EIP)

## Why Elastic IP exists
Public IPs change when EC2 is stopped and started.

## What is an Elastic IP?
A static public IP address provided by AWS.

## Key features
- Does not change on stop/start
- Can be reassigned to another EC2
- Charged when unused

## When to use
- Production servers
- Failover scenarios

## Real-world analogy
Elastic IP is a permanent phone number.
