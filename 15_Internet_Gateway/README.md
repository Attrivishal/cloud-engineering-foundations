# Internet Gateway (IGW)

## Why IGW exists
AWS networks are isolated by default.

## What is an Internet Gateway?
An Internet Gateway connects a VPC to the internet.

## What IGW allows
- Inbound internet traffic
- Outbound internet traffic

## Required for
- Public subnets
- Public-facing EC2 instances

## Diagram
VPC → Internet Gateway → Internet

## Interview explanation
"An Internet Gateway allows resources in a VPC to communicate with the internet."
