# Networking Fundamentals

## Why this topic exists
Servers communicate using networks.

## IP Address
An IP address uniquely identifies a server.

## Private vs Public IP
- Private IP: internal communication
- Public IP: internet access

## Ports
Ports identify applications.
22 = SSH, 80 = HTTP, 443 = HTTPS

## Real-world analogy
IP = house address  
Port = door number  

## Diagram
User → IP → Port → Application

## Interview explanation
"Clients connect to servers using an IP address and a port."
