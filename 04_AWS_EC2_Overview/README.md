# AWS EC2 Overview

## What is EC2?
Amazon EC2 is AWS’s service for providing virtual machines in the cloud.

## What an EC2 instance is
An EC2 instance behaves like a traditional server but is hosted on AWS.

## Why EC2 is used
- Launch servers quickly
- Scale easily
- Pay only for usage

## Real-world example
Launching a Linux server in minutes instead of waiting weeks for hardware.
