# Cloud & EC2 Fundamentals (Interview + Job Ready)

This repository is a beginner-to-intermediate guide to understanding
AWS EC2 and core cloud fundamentals from first principles.

### Who is this for?
- Cloud / AWS freshers
- Students preparing for interviews
- Beginners confused by cloud concepts
- Anyone who wants strong fundamentals

### How to use this repo
- Read folders in numerical order
- Each folder contains one focused topic
- Content is written in interview-ready language
- Practice explaining concepts aloud

### Topics covered
- Servers, Virtualization, EC2
- Networking & Security
- Storage & AMI
- Public vs Private Subnets
- NAT, Elastic IP
- Real troubleshooting scenarios

This repo focuses on **understanding**, not memorization.
